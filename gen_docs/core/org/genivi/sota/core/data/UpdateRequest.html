UpdateRequest

Related Docs: object UpdateRequest | package data

case class UpdateRequest(id: UUID, namespace: Namespace, packageUuid: UUID, creationTime: Instant, periodOfValidity: Interval, priority: Int, signature: String, description: Option[String], requestConfirmation: Boolean, installPos: Int = 0) extends Product with Serializable

Domain object for an update request. An update request refers to the intent to update a package on a number of devices. It describes a single package, with a period of validity and priority for the update.

id

A generated unique ID for this update request

creationTime

When this update request was entered into SOTA

periodOfValidity

The start and end times when this update may be installed. The install won't be attempted before this point and will fail it it hasn't been started after the interval expires.

priority

The priority. priority == 1 items are installed before priority == 2.

signature

Signature for this updates Id

description

A descriptive text of the available update.

requestConfirmation

Flag to indicate if a user confirmation of the package is required.

installPos

Derived from the corresponding field in UpdateSpec
