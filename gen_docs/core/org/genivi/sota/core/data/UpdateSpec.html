UpdateSpec

Related Docs: object UpdateSpec | package data

case class UpdateSpec(request: UpdateRequest, device: Uuid, status: UpdateStatus, dependencies: Set[Package], installPos: Int, creationTime: Instant, updateTime: Instant) extends Product with Serializable

A combination (campaign, device, packages) --- which remain constant over time --- along with the latest UpdateStatus (for that campaign on this device) --- which may change over time.
An UpdateSpec never gets deleted, just its status is rewritten. Upon reaching Finished a row is added to InstallHistoryTable to record that fact

  • The campaign (ie, UpdateRequest) that initiated this UpdateSpec is linked from here.
  • Among pending UpdateSpec targeting a device an installation order is tracked by installPos
